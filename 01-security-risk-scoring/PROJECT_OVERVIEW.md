# AI Security Risk Scoring Model

## Purpose

This project demonstrates how Artificial Intelligence and Machine Learning can support:

- Intelligence Analysis
- ISSO Risk Management
- Insider Threat Detection
- Personnel Security
- Investigations

## Scenario

A cleared organization wants to identify high-risk activity by analyzing:

- Failed login attempts
- After-hours access
- Unusual file access
- Policy violations
- Privileged account activity

## Machine Learning Model

This project uses:

- Python
- pandas
- scikit-learn
- Decision Tree Classification

The model predicts whether an event is:

- Low Risk
- Medium Risk
- High Risk

## Real World Applications

This model can be adapted for:

- Insider Threat Programs
- Personnel Security Investigations
- Continuous Monitoring
- RMF Risk Assessments
- Cyber Threat Intelligence
- User Activity Monitoring
