# Security Risk Scoring Model

## Project Overview
This project uses Python and machine learning concepts to score fictional security events as Low, Medium, or High Risk.

The project connects AI/ML with intelligence analysis, ISSO responsibilities, insider-threat awareness, and investigative decision-making.

## Scenario
A cleared organization wants to identify high-risk activity based on user behavior, access activity, failed login attempts, after-hours access, and policy violations.

## Skills Demonstrated
- Python
- scikit-learn
- Data analysis
- Security risk scoring
- ISSO mindset
- Investigative analysis
- Insider threat indicators

## Sample Risk Factors
- Failed login attempts
- After-hours access
- Unusual file access
- Policy violations
- Privileged account activity
- Prior security incidents

## Goal
Build a simple machine learning model that predicts whether an activity is Low, Medium, or High Risk.
