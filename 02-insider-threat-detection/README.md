# TensorFlow Insider Threat Detection Model

## Project Overview

This project uses TensorFlow to identify potential insider-threat activity using fictional security data.

## Scenario

A cleared organization wants to detect suspicious user behavior that may indicate insider-threat risk.

## Risk Indicators

- Failed login attempts
- After-hours access
- Large file downloads
- Unusual file access
- Policy violations
- Privileged account use
- Prior security incidents

## Model Goal

The AI model predicts:

- Normal
- Suspicious
- High Risk
