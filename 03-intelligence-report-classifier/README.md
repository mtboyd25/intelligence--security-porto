# PyTorch Intelligence Report Classifier

## Project Overview

This project uses PyTorch to classify fictional intelligence reports into different intelligence disciplines.

## Intelligence Categories

- HUMINT (Human Intelligence)
- GEOINT (Geospatial Intelligence)
- SIGINT (Signals Intelligence)
- OSINT (Open Source Intelligence)
- MASINT (Measurement and Signature Intelligence)

## Scenario

An intelligence organization receives reports from multiple sources and wants to automatically classify each report according to the intelligence discipline it belongs to.

## Skills Demonstrated

- Python
- PyTorch
- Natural Language Processing
- Intelligence Analysis
- Machine Learning
- Investigative Analysis

## Goal

Train a PyTorch model to classify intelligence reports into the correct intelligence category.
